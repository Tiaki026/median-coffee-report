# Median Coffee Report 
## Отчет о потреблении кофе.

## :page_with_curl: Описание
Скрипт читает csv-файлы с данными о подготовке студентов к экзаменам и формирует один отчет в котором будет медианная сумма трат на кофе по каждому студенту за весь период сессии.
Отчёт включает в себя список студентов и медиану трат (по колонке coffee spent), отчёт сортируются по убыванию трат. Название файлов, может быть несколько, передается в виде параметров --files и название отчета --report (в нашем случае это median-coffee). Отчёт формируется по всем переданных файлам, а не по каждому отдельно.
Проект покрыт тестами в количестве 10 штук.
